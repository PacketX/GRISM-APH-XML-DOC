# Ethernet Port Link LED

This document describes the link LED indications for Ethernet ports.

## Link Speed vs LED Status

| Link Speed | LED Status |
|---|---|
| 10G | **Solid Green** - Linked at 10G.<br>**Blinking Green** - Activity at 10G, 5Hz. |
| 25G | **Solid Cyan** - Linked at 25G.<br>**Blinking Cyan** - Activity at 25G, 5Hz. |
| 40G | **Solid Blue** - Linked at 40G.<br>**Blinking Blue** - Activity at 40G, 5Hz. |
| 100G | **Solid White** - Linked at 100G.<br>**Blinking White** - Activity at 100G, 5Hz. |
